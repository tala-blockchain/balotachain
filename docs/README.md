# docs

Developer and operator documentation for BalotaChain.

| Subdirectory | Contents |
|---|---|
| [`adr/`](adr/) | Individual architecture decision records, Michael Nygard format. Process documented in [`adr/README.md`](adr/README.md). |
| [`architecture/`](architecture/) | Higher-level architecture documents covering broad commitments that span multiple ADRs. |

Per-issue documentation lands here as work progresses:

- `dev-environment.md` — detailed per-OS developer setup (issue #3).
- `branch-policy.md` — branch protection summary (issue #6).
- Operator runbooks for trustee and admin roles (later issues).

Protocol specifications and threat model artifacts belong under [`../spec/`](../spec/), not here.
